I watched all unreal engine live training youtube videos by epic games to try and get a broader understanding of the engine and other miscellanous 3D concepts. I took notes from each video on what I learned or key parts of the video.

Introduction

I've been wanting to learn unreal engine for quite some time and first time I briefly touched the engine was in 2015 with version 4.7. Where I tinkered with the engine for maybe 60 hours in total. I did also touch UDK back in ~2012 but never got very far with unrealscript. But now in May 2017 I decided to give it another go and came across a live training stream Epic has been running about once a week for few years now. The videos I watched are saved to youtube in this youtube playlist. I decided to try and watch all of the videos to better learn about the engine and other concepts I might never have heard about before.

I decided to watch 1 video a day and started around 8th of May and finished on September the 16th. Watching videos with 1.25x speed saves a lot of hours :). I think I saved about 40 hours by using 1.25x speed over 100 videos. Some videos were far more interesting than others but for example most of the things Alan Willard did with materials using displacement or world offsets were quite cool. I also did not always watch the videos with 100% focus as not everything was super interesting to me or I already knew a fair bit about the subject. Videos about VR became a history timeline about Oculus, Samsung Gear VR and eventually Vive.

The notes I took was mainly for myself incase I need to search on an issue to find a possible video touching on the subject. However I figured that these notes could serve as a secondary overview and thus decided to write this blogpost. Also I wrote a C# console application to parse the CSV file into HTML output because editing 90 rows by hand would have been quite cumbersome task! :)

C# CSV Parser source code

This is the source code for the C# console application to automatically generate the HTML code. It parses a csv exported by google documents using the TextFieldParser from Microsoft.VisualBasic.dll 

        
namespace CSVParser
{
    class Program
    {
        static void Main(string[] args)
        {
            List<string[]> rows = GetRows("uelivetraining.tsv");
            string output = CreateOutput(rows);
            File.WriteAllText("output.txt", output);
        }
        
        static List<string[]> GetRows(string a_filename)
        {            
            bool firstParse = false;
            List<string[]> rows = new List<string[]>();
            using (TextFieldParser parser = new TextFieldParser(a_filename))
            {
                parser.TextFieldType = FieldType.Delimited;
                parser.SetDelimiters("\t");
                while (!parser.EndOfData)
                {
                    //Process row
                    string[] fields = parser.ReadFields();
                    if (!firstParse)
                    {
                        firstParse = true;
                        continue;
                    }
                    rows.Add(fields);                    
                }
            }
            return rows;
        }
        static string CreateOutput(List<string[]> a_rows)
        {
            string output = "";
            StringBuilder outputBuilder = new StringBuilder();
            outputBuilder.AppendFormat("<dl>" + Environment.NewLine);
            foreach (string[] row in a_rows)
            {
                string nameUnderscored = row[0].Replace(' ', '_');
                // For the ordered list inside the <details> tag
                //outputBuilder.AppendFormat(
                //    "<li>" + Environment.NewLine +
                //        "\t<a href=\"#{0}\">{1}</a>" + Environment.NewLine +
                //    "</li>" + Environment.NewLine, nameUnderscored, row[0]);
                outputBuilder.AppendFormat(
                    "\t<dt>" + Environment.NewLine +
                        "\t\t<a name=\"{0}\">{1}</a>" + Environment.NewLine +
                    "\t</dt>" + Environment.NewLine +
                    "\t<dd>" + Environment.NewLine +
                        "\t\t{2}" + Environment.NewLine +
                    "\t</dd>" + Environment.NewLine
                    , nameUnderscored, row[0], row[1]);
            }
            outputBuilder.AppendFormat("</dl>" + Environment.NewLine);
            output = outputBuilder.ToString();
            return output;
        }
        
    }
}
